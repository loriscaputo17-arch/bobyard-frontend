# Bobyard Comments — Frontend

React app for a comment system with admin/user roles.

**Backend repo:** https://github.com/loriscaputo17-arch/bobyard-backend
**Live app:** https://bobyard-frontend.vercel.app

## Stack
- React + Vite
- Tailwind CSS
- React Router

```bash
cd frontend
npm install
```
